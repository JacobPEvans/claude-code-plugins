# git-standards

Git workflow standards, branch hygiene, PR creation guards, and issue linking.

## Skills

- **`/git-workflow-standards`** - Branch management, worktrees, merge conflicts, syncing
- **`/pr-standards`** - PR guards, issue linking, GitHub issue templates, AI mention rules

## Installation

```bash
claude plugins add jacobpevans-cc-plugins/git-standards
```

## Usage

```text
/git-workflow-standards
/pr-standards
```
