# git-troubleshooting

Claude Code plugin for diagnosing and recovering from git rebase failures, pre-commit hook issues, and worktree problems.

## Skills

- **`/git-rebase-troubleshoot`** - Diagnose and recover from rebase failures including push rejections and conflicts
- **`/git-precommit-troubleshooting`** - Fix pre-commit hook auto-fix loops and staged file mismatches
- **`/git-worktree-troubleshooting`** - Resolve worktree path issues, ambiguous refnames, and branch problems

## Installation

```bash
claude plugins add jacobpevans-cc-plugins/git-troubleshooting
```

## Usage

```text
/git-rebase-troubleshoot
/git-precommit-troubleshooting
/git-worktree-troubleshooting
```
